git init - iniciar um repositorio local (maquina do programador) branch - são os nomes de ramificações em um projeto, onde cada desenvolvedor pode criar a sua (raiz) do projeto principal e fazer alterações de forma individual sem alterar o arquivo principal assim conseguindo diminuir as ocorrencias de erro ou apagar o codigo do colega, sendo assim os alunos podem desenvolver a sua parte do projeto de forma simutanea sem preescrever o desenvolvimento do proximo. git branch -M main - Modificar o nome da branch principal de master para main. git remote add origin - Faz um vinculo do repositorio local com o repositorio remoto. git config --global user.name " " - Configurar o nome de usuario do GitHub git config --global user.email " " - Configurar o email de usuario do GitHub git status - Verificar o status dos arquivos git add - Adicionar o arquivo ao staging area git commit -m " " - Adicionar comentarios sobre a sua mudança
